# Custom Agno Agents

This directory contains our custom-built Agno agents organized by domain and purpose.

## Directory Structure

- `financial/` - Financial analysis and market research agents
- `research/` - Research and data analysis agents
- `creative/` - Creative content generation agents
- `utils/` - Shared utilities and helper functions for our agents

## Getting Started

Each agent directory contains:
- Agent implementation files
- Configuration files
- Documentation
- Example usage

## Best Practices

1. Keep agents focused and specialized
2. Document all tools and capabilities
3. Include example usage and test cases
4. Version your agents independently
5. Share common utilities in the `utils/` directory

## Contributing

When adding new agents:
1. Create a new directory in the appropriate category
2. Include a README.md with usage instructions
3. Add example code and test cases
4. Document any dependencies or requirements
